# sesi_lop_funcionarios_vpf02_2026

Este repositório possui dois programas em C com a seguinte estrutura:

``` C
struct Funcionario{
	char nome[50];
	char cargo[30];
	int salario;
};
```
- O primeiro funcionarios.c cria um arquivo de texto chamado funcionarios.csv com 5 funcionários cadastrados
- O segundo lê o arquivo funcionarios.csv e faz um relatorio de quantos funcionários têm na empresa e a média salarial da empresa

# Tecnologias
- Dev C++

# Passos para testar
Abra os arquivos .c com Dev C++ e execute.
